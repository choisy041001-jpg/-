## 1. 서비스 플로우

```mermaid
flowchart TD
    Start([시작]) --> Step1[단계 1]
    Step1 --> Cond1{조건 확인}
    Cond1 -->|성공| Step2[단계 2]
    Cond1 -->|실패| Error[오류 처리]
    Step2 --> Step3[단계 3]
    Step3 --> End([종료])
    Error --> End
```

---

## 2. 예외 플로우

flowchart TD

    B -->|케이스 2| D[예외 처리 2]
